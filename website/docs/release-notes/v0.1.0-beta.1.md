# v0.1.0-beta.1

The first Gitmoot beta focuses on local-first coordination:

- local SQLite workflow state
- GitHub PR comment routing
- daemon polling
- agent startup and subscription
- Codex, Claude Code, and shell runtime adapters
- presets for planner and review agents
- plugin packaging for Codex and Claude Code
- branch locks, jobs, status, and smoke-test documentation

For current release artifacts, see
[GitHub releases](https://github.com/jerryfane/gitmoot/releases).
